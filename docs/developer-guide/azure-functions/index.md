# Azure Functions Developer Reference
{: .no_toc }

Configuration and APIs for Durable Functions running in Azure Functions.
{: .fs-6 .fw-300 }

---

## Overview

When running Durable Functions in Azure Functions, you have access to additional configuration options and built-in HTTP APIs for managing orchestrations. This section covers:

- **host.json Settings** - Configure extension behavior, performance tuning, and storage providers
- **Bindings** - Orchestration, activity, entity triggers and the durable client binding
- **HTTP API** - Built-in REST APIs for instance management
